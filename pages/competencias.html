Competencias Profesionales

Mi trayectoria profesional y académica me ha permitido desarrollar competencias clave en bioinformática, investigación y desarrollo de software aplicado a la biología. Estas competencias se agrupan en las siguientes áreas:

  • Bioinformática y Genómica: Análisis de datos de secuenciación masiva, ensamblaje de genomas, identificación de variantes y estudio de metilación del ADN.
  • Desarrollo de Software Científico: Programación en Python, R, Bash y SQL para la automatización y optimización de análisis bioinformáticos.
  • Aprendizaje Automático: Aplicación de modelos de machine learning para la clasificación de imágenes biológicas y el análisis de datos genómicos.
  • Administración de Sistemas y HPC: Gestión de entornos Linux, configuración de clústeres de computación de alto rendimiento (HPC) y uso de gestores de tareas como SLURM.
  • Visualización de Datos: Creación de gráficos y dashboards interactivos para la interpretación de resultados científicos.

Competencias Digitales

El dominio de herramientas digitales es clave en la investigación actual. Poseo experiencia en:

  • Manejo de herramientas de control de versiones como Git y GitHub.
  • Desarrollo de workflows bioinformáticos con Snakemake y Nextflow.
  • Uso de entornos de desarrollo como Jupyter Notebook y RStudio.
  • Gestión y análisis de grandes volúmenes de datos en la nube y en servidores HPC.
